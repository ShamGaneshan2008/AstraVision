02 / CAPABILITIES
PLATFORM

Engineered for planet-scale inference.

A composable observability stack — from raw radiance to ranked alerts — built on open standards and battle-tested at terabyte scale.

01

Multi-sensor fusion

Harmonised Landsat, Sentinel-1/2 and MODIS into a single temporal cube with cloud-aware compositing.

02

Sub-pixel co-registration

Phase-correlation alignment locks scenes to ±0.3 px so detections aren't fooled by orbital drift.

03

Vision Transformer detector

A Siamese ViT trained on 14M labelled scene-pairs across six biomes and four climate zones.

04

Streaming GeoJSON

Detections push as RFC-7946 features over WebSocket — drop straight into Mapbox, Deck.gl or QGIS.

05

Calibrated confidence

Per-pixel Bayesian uncertainty, not just softmax — so downstream alerts can be ranked by risk.

06

Replayable history

Run any AOI back to 2015 with deterministic outputs and full provenance for audit and ESG reporting.

main.py — orbital_ai ● LIVE 
from fastapi import FastAPI, WebSocket
from orbital.detect import SiameseChangeNet
from orbital.sources import STACClient

app = FastAPI(title="Orbital.AI")
model = SiameseChangeNet.load("v2.4")

@app.post("/detect")
async def detect(aoi: AOI, t0: date, t1: date):
    scenes = await STACClient.search(aoi, t0, t1)
    delta  = await model.predict(scenes)
    return {
        "confidence": delta.score,
        "geojson":    delta.to_geojson(),
        "classes":    delta.classes,
    }
07 / FAQ
ANSWERS

Everything you'd ask on day one.

What's the smallest change Orbital.AI can detect?

At Sentinel-2 native 10 m resolution we reliably surface contiguous changes ≥ 0.05 ha (a tennis court). With SAR fusion we push that to 0.02 ha for clearings with strong dielectric contrast.

How quickly are new scenes processed?

Median end-to-end latency from data-take to detection is 17 minutes for Sentinel-2 and 41 minutes for Landsat 9. Alerts are pushed over WebSocket as soon as they clear the confidence threshold.

Can I bring my own model?

Yes. Any TorchScript or ONNX model with a (T, C, H, W) → (H, W, K) signature drops into the detect stage. The pipeline still handles ingest, alignment, calibration and serving.

What about cloud cover?

Each scene is masked with a CNN-based cloud and shadow detector. SAR data fills optical gaps, and we compute confidence with cloud-fraction priors so noisy windows degrade gracefully instead of failing silently.

Is the data licensable for commercial use?

Landsat, Sentinel and MODIS are all open data. Our derived GeoJSON detections are released under your tenant's chosen licence — including fully proprietary on private deployments.
